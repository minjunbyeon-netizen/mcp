# 🤖 블로그 글 '인간미 강화' 업데이트 (2026-02-26)

블로그 글이 AI처럼 느껴지는 문제를 해결하기 위해, 시스템 전반에 문장 다양성 및 서술적 뉘앙스를 주입하는 **'인간화(Humanization)'** 로직을 적용했습니다.

## 🛠️ 주요 변경 사항

### 1. 페르소나 설정 확장 (`persona_version_manager.py`)
각 페르소나 설정에 인간미를 조절하는 4가지 핵심 파라미터가 추가되었습니다.
- **Narrative Flow:** 서술 방식 고도화 (Structured, Flexible, Storytelling, Chatty)
- **Personal Insight:** 보도자료 외 주관적 의견/느낌 비중 추가 (10% ~ 70%)
- **Human Catchphrases:** 페르소나 고유의 말버릇 적용 (예: "그거 아세요?", "사실 말이죠")
- **Anti-AI Cliches:** AI 모델이 즐겨 쓰는 상용구 자동 제거 (예: "혁신적인", "다각도", "통해")

### 2. 프롬프트 엔진 고도화 (`run_blog_generator.py`)
- **버스트니스(Burstiness):** 문장 길이를 의도적으로 짧고 길게 섞어 리듬감을 형성하도록 강제.
- **주관적 소회 주입:** 보도자료의 팩트 전달 사이에 페르소나의 직업적 소견을 반드시 포함하도록 명령.
- **구어체 연결어:** 딱딱한 문어체 대신 자연스러운 구어체 사용 비중 상향.

### 3. 문장부호 및 기호 정밀 정화
- **기호 오남용 제거:** `.,`, `;;`, `!!`, `??` 등 AI가 습관적으로 만드는 지저분한 기호 조합을 금지했습니다.
- **기호 다이어트:** 
    - 이모지는 문단 끝에만 최소화(1개 이내).
    - 강조 시 따옴표(`" "`) 사용을 자제하고 **굵게(Bold)** 처리 권장.
    - 문장 끝 외의 불필요한 마침표(`.`) 사용 억제.

## 📝 사용 및 테스트 방법

### 기존 페르소나 (v1 등)
1. `run_blog_generator.py`를 실행하여 블로그를 생성합니다.
2. 생성 후 피드백 단계(4점 이하)에서 **"말투/어미가 안 맞아요"** 또는 다른 항목을 선택하면 자동으로 새 버전(`v2`...)이 생성되며 위 설정들이 주입됩니다.

### 신규 페르소나
- 앞으로 생성되는 모든 페르소나는 격식도에 맞춰 이 '인간화 설정'이 자동으로 포함되어 생성됩니다.

---
**업데이트 결과:** 기계적인 문장 반복이 줄어들고, 실제 담당자가 자기 생각을 섞어서 쓴 것 같은 자연스러운 글이 생성됩니다.
